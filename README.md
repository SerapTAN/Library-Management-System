# Library-Management-System
Java console application for managing library books, including adding, searching, borrowing, and returning books.

## Description
This project is a Java console-based Library Management System that allows users to manage basic library operations. The system supports adding books, searching for books, borrowing, and returning them. The project is designed to demonstrate object-oriented programming principles and clean program structure using Java.

## Features
- Add new books to the library
- Display available books
- Search for books by title or ID
- Borrow and return books
- Input validation for user actions
- Simple and interactive console interface

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Classes and Objects
- Conditional statements and loops
- Standard Input/Output

## How to Run
1. Compile the program: javac Main.java

## Example
**User Actions:**
- Add a new book  
- Search for a book by title  
- Borrow a book  

**Output:**
- Book added successfully  
- Book found  
- Book borrowed successfully
